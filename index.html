⚙️ One-time Supabase setup needed To enable live cross-user reports, create a free Supabase project and add your credentials to the top of this file.
  1. Go to supabase.com → New project (free tier)
  2. In the SQL editor, run the setup query shown in the <!-- SUPABASE SETUP --> comment at the top of index.html
  3. Copy your Project URL and anon public key from Project Settings → API
  4. Paste them into SUPABASE_URL and SUPABASE_KEY at the top of the <script> block
  5. Enable Row Level Security → add a policy allowing SELECT and INSERT for anon role
Until then, reports are saved to this browser only.
